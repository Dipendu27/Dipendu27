📈 Featured Project

🌐 30-Day DevOps Cloud Testing Journey￼

<p align="center">
  <img src="https://img.shields.io/badge/Project-Cloud%20Testing%20Journey-0E75B6?style=for-the-badge" />
  <img src="https://img.shields.io/badge/Stack-Docker%20%7C%20GitHub%20Actions%20%7C%20AWS-success?style=for-the-badge" />
</p>

A capstone repository demonstrating environment independence, CI/CD execution, and secure cloud-based telemetry tracking for infrastructure testing.

* Dynamic Data Parsing: Built Bash scripts to parse live e-commerce data arrays using jq loops
* Docker Multi-Tool Environment: Containerized test workflows with Alpine Linux to eliminate configuration drift
* CI/CD Orchestration: Created a GitHub Actions pipeline to run Newman collections automatically and track logs
* Cloud Observability: Routed secure telemetry logs from AWS EC2 into Amazon S3 using IAM-based access controls
